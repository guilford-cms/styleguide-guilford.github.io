Rich Text

The rich text component is a WYSIWYG that allows for the insertion of one media element that can be floated within the text either left or right. The component is used on both the page template (with a 10 column layout) and the news/story template (with an 8 column layout). The user can insert a maximum of one item from the 5 options below.
