Factoid

Factoids are available for use on the page template to display facts. Similar in functionality to a smart teaser, the layout of the blocks will change as more are added. There is a maximum of 4 blocks per component.

Block background color User can pick from these four options:

.bg-oscar
.bg-bert
.bg-elmo
.bg-light-grey
