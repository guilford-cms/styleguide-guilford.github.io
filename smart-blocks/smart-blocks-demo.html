Smart Blocks

Smart blocks will allow the user to enter a minimum of 3 cards and a maximum of 5 cards on the page template. As the user enters blocks, the layout of the component will change accordingly. The height of the row is determined by the tallest card.  Additional body copy is only revealed on hover (Desktop Only).
