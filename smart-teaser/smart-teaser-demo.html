Smart Teaser

The smart teaser will allow the user to enter a maximum of 4 blocks on to the landing/page template. As the user enters blocks, the layout of the component will change accordingly. The media block supports image, video and galleries. The ratio of the thumbnail is 16x9. All of the fields for this component are optional, BUT when using a field in the first block and the user enters another block, that field becomes manadatory. For example, if the user enters media is block A…Blocks, B, C & D all needs to have the media field populated. The only exception to this rule is the button.
